# Assets Directory Structure

This folder contains all game assets organized by type.

## Structure

```
assets/
├── images/
│   ├── ui/          # UI elements (buttons, panels, frames)
│   ├── icons/       # Icon assets
│   ├── backgrounds/ # Background images
│   └── sprites/     # Game sprites and textures
├── fonts/           # Custom fonts (.woff, .woff2, .ttf)
├── audio/
│   ├── sfx/         # Sound effects
│   └── music/       # Background music
└── (css files)      # Stylesheets
```

## Usage

Import assets using the `@/assets/` path alias.
